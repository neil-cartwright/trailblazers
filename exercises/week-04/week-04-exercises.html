Trailblazers Week Four

Once you have opened your browser window and notepad, please try to split the screen so that half is code, and the other half is browser. Then you will see your changes update quicker.

Please remember to save your code after each edit ctrl + s, and refresh the browser window ctrl + r - otherwise your edits will not be displayed

Exercise One | :hover styles

I am a 'div' element.

I have an 'id' of 'box'.

Hover the mouse over me to display the :hover styles

Find the styles relating to #box. Try changing some of the hover styles.

To begin with try changing the background-color.

You can change/add any styles. Try to change the width and height

Here are some styles you could try to add to the :hover pseudo-class:

  • width:200px;
  • font-family:courier;
  • background-color: pink;
  • border-radius:50%;
  • font-size:50pt;
  • text-transform:uppercase;
  • margin-left:50px;
  • padding:50px;
  • cursor:cell;

find out more at this website: W3C Schools

Exercise Three | Images and Mouseovers

I am an img element.

If you onmouseover the image you will invoke a javascript function called changeImage()

When you onmouseout of the image it will invoke a function called changeImageBack()

Find the img tag. Try changing the src="" and change the default pictures. Look in the images folder (directory) for an alternative image.

Advanced Try to change the javascript. Instead of using an onmouseover event try using an onclick event. Instead of onmouseout try using ondblclick (on double click).
