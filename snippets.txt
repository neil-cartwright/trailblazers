Exercises:

<!-- set up the document -->

1. open basic.html in notepad.

2. open basic.html in a web browser.

  ! the screen will be blank until you add some content !


<!-- add some snippets -->

3. Add this snippet between the <body></body> tags

    <div class="myHeader">

      <h1>My Web Page</h1>

    </div>


    -- then save (ctrl + s) and refresh the browser (ctrl + r) --


4. Add this snippet between the <head></head> tags, underneath the <title></title> tag:

    <style>
      body {
        font-family:'comic sans ms';
      }

      .myHeader {
        text-align:center;
      }
    </style>

    -- then save and refresh the browser --

5. Add this snippet underneath the <div class="myHeader"></div> tag

  <div class="myPicture">

  </div>


6. Add this snippet in the <style></style> tags, underneath the .body styles.

    .myPicture {
      height: 50vh;
      background:url('images/beard.png');
      background-size:contain;
      background-position:center;
      background-repeat:no-repeat;
    }

    -- save and refresh the browser --

7. Go to the images folder and pick a picture that you would like to show on the screen. Change the following snippet to reflect your choice:

  background:url('images/your-picture-choice.png');


8. Make your <h1 class="myHeader"></h1> Header bigger by adding the following style to your stylesheet in the correct place:

  font-size:36pt

  -- save and refresh the browser --
